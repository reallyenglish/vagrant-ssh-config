# Vagrant::SSH::Config

Access `vagrant ssh-config` from ruby code.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'vagrant-ssh-config'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vagrant-ssh-config
